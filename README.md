# XMUT-travel_analysis
小学期作业：https://www.kaggle.com/code/devraai/travel-trip-records-analysis-and-duration-predi/notebook 
## 主要功能： 
1) 数据导入 给定 csv文件，将它导入到数据库中
2) 统计图数据可视化 对比不同性别的旅行周期
3) 预测 根据 Traveler age，Accommodation cost 和 Transportation cost 预测旅行周期。 
## 新增功能 
a)	多维度可视化（5 类对比分析 + 交互式功能）  
b)	旅行周期预测（3 个输入参数 + 预测区间 + 历史记录）  
c)	旅行费用计算器（总预算 + 费用占比分析）
